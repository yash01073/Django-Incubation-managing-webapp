My Django app is based on managing incubation and training facility 
related managing facilities made for the purpose of an incubation cell.


This consist of
1. Signup and login forms for training and incubation
2. Payment portal
3. Dashboard for the status of the availability of training rooms
4. Dashboard for the user information
and other site related pages foe general information


Technology used: Python, Django , Css, Javascript, Html,
                 Session creating
Database used: Mysql

This project is still under development.
